## 🛠️ Common Tasks

### Building
```powershell
# Build Windows Forms (Desktop)
dotnet build Calculator.csproj --configuration Release

# Build Web Application
dotnet build CalculatorWeb.csproj --configuration Release

# Copy web assets
Copy-Item -Path "wwwroot\*" -Destination "bin\Release\net48\wwwroot\" -Recurse -Force
```

### Running
```powershell
# Run Desktop version
.\bin\Release\net48\Calculator.exe

# Run Web version
.\bin\Release\net48\CalculatorWeb.exe
# Then open: http://localhost:5000
```

### Deployment
```powershell
# See DEPLOYMENT_GUIDE.md for detailed instructions

# Quick deploy to IIS
Copy-Item -Path "bin\Release\net48\*" -Destination "C:\inetpub\wwwroot\Calculator\" -Recurse -Force
```

### GitHub Actions
```bash
# Push to trigger workflow
git push origin main

# View in GitHub
# Go to: Repository → Actions tab
```
